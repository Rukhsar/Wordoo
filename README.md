
# Wordoo
Bootstrap 3 based Boilerplate for creating quick wordpress themes, using:

- Sass
- Browsersync
- HTML Templates
- CSS, JS Builder + Minify + Uglifyjs
- Image optimization

## Installation
- You need Node.js, Bower and Yarn installed.

```
 yarn install
 gulp
 open http://localhost:9001 in your browser
```
